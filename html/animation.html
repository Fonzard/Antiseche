Dis les termes

@keyframes

La règle @keyframes permets de définir les étapes qui compose la séquence d'une animation CSS

Transform

Transform nous permet d'assigner une ou plusieurs effets de transformation à appliquer à un élément : rotate / translate / scale / skew / matrix

Animation

Les animations vont permettre de gérer le comportement des animations CSS en définissant la durée, le nbre de répétition, le comportement de répétition.

Pour définir une animation sur css, il faut utiliser les propriétés de type animation- name / duration / timing-function / delay / iteration-count / direction / fill-mode / play-state. on peut aussi utiliser le raccourci animation

Opacity 0 to 1

@keyframes colorChange{   
                    0% {               
                    opacity:0;         
                    }                  
                    100% {             
                    opacity:1;        
                        }

#opacity div {
            animation-name:colorChange;
            animation-duration:3s;
            animation-delay:2s;
            animation-fill-mode:forwards;
            }

Transform:translate

@keyframes moveRight{                  
                0% {                                
                    transform:translateX(0);        
                }                                   
                100% {                              
                    transform:translateX(15vh);   
                }

#transform-translate div {
                    animation-name:moveRight;
                    animation-duration:3s;
                    animation-delay:2s;
                    animation-fill-mode:forwards;
                }

Translate / rotate

@keyframes rolledRight{
                    0% {
                        transform:translateX(0) rotate(0deg);
                    }
                    100% {
                        transform:translateX(15vh) rotate(180deg) ;
                    }

#translate-rotate div {
                        animation-name:rolledRight;
                        animation-duration:3s;
                        animation-delay:2s;
                        animation-fill-mode:forwards;
                    }
