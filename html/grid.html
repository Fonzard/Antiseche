.parent { display:grid;} .simple {grid-template-rows:(2,1fr); grid-template-columns:(8,1fr);} .item1 {grid-area: 1 / 1 / 2 / 2;} .item2 {grid-area: 2 / 1 / 3 / 2;} .item3 {grid-area: 1 / 2 / 7 / 8;} .item4 {grid-area: 2 / 2 / 8 / 8;}

On definit le type d'affichage sur le parent display:flex; puis on définit le nombre et la taille des lignes / colonnes grid-template-rows / grid-template-columns. Pour finir on définit la taille et l'emplacement des items grid-area: row-start / column-start / row-end / column-end

Grid items

Explication

.grid-item {
                display:grid;
                grid-template-areas:"header  header  header   header"
                                    "main    main    main     sidebar"
                                    "footer  footer  footer   footer";
                grid-template-rows: 20% auto 20%;
                gap:1rem;
             }

.grid-item > .gheader {grid-area:header;}

.grid-item > .main {grid-area:main;}

.grid-item >
.sidebar {grid-area:sidebar;}

.grid-item > .gfooter {grid-area:footer;}

Grid area

Explication

.grid-area {
                display:grid;
                grid-template-columns: repeat(4,1fr);
                grid-template-rows:auto;
                gap:1rem;
                }

.grid-area > .gheader { grid-area: 1 / 1 / 2 / 5;}

.grid-area > .main { grid-area: 2 / 1 / 4 / 4;}

.grid-area > .sidebar { grid-area: 2 / 4 / 4 / 5;}

.grid-area > .gfooter { grid-area: 4 / 1 / 5 / 5;}
